# Native Apps

## iOS (v1.4.0)

4 tabs: Situation (map) · Markets · People · Settings

- **Situation:** 8 live map layers, MapKit, location-aware
- **Markets:** Stocks/crypto/commodities, inline portfolio (net worth, debt, payday), Fear & Greed Index, stock detail sheets with 9 chart types
- **People:** Google → DuckDuckGo → Wikipedia cascade search, social links, indexed profiles
- **Settings:** Avatar, subscription tier, Tally credentials, map layer toggles

Platform: iOS 17+, portrait-only, dark mode only, PWA companion

## macOS (v0.6.1)

5 tabs: Situation · Markets · Portfolio · Settings · People (Cmd+1–5)

- **Situation:** Same 8 map layers as iOS
- **Markets:** Sortable table with search, stock detail with chart scrub + stats grid
- **Portfolio:** 6 subtabs — Spending (forecast, income scenarios), Holdings, Budget, Debt (avalanche/snowball), Goals, Statements
- **Settings:** Profile, tier, Tally, map layer toggles
- **People:** Same search as iOS

Platform: macOS 14+, resizable (default 1200×800), dark mode only

## watchOS (v1.0.0)

Complications-focused. Minimal UI. Shows market snapshots and alerts.

## Widgets

- **iOS Widgets:** Portfolio (net worth), Markets (ticker snapshot), Watchlist — WidgetKit
- **macOS Widgets:** Portfolio, Markets — WidgetKit
